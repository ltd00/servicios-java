Desarrollo para Sistemas Distribuidos

Objetivo del proyecto

El objetivo del proyecto es implementar una arquitectura SOA que de soporte al proceso de negocio de pedidos de suministros.
Beneficios del proyecto

Composición: En el campamento minero se utiliza el modulo de “generación de pedidos” y en la sede de lima el “modulo de compras” para realizar el proceso de compra.

Agilidad: La arquitectura permitirá agilizar el flujo del proceso de pedidos debido a que no es necesario enviar correos o hacer llamadas telefónicas para saber el estado de compra del pedido.

Reusabilidad: Desde cualquier sede de producción se puede solicitar el servicio de compras y estado de atención del pedido
